解决.gitignore过滤规则不起作用问题

天赐 · 2019-7-22 · 次阅读

问题

在Git使用.gitignore进行过滤时不起作用

解决

# 最后是一个"."
git rm -r --cached .

如果出现：use -f to force removal，则使用git rm -rf --cached .
